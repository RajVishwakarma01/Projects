About Us




Searching for a parking spot has become a daily habit for a lot of people all around the globe. It is usually a time consuming and frustrating process that a lot of citizens have to go through.
This ParkSmart based on Slot booking is implemented, using the Android application. It is an efficient one for solving parking problems, which overcomes the traffic congestion also provides automated billing process.

Introducing ParkSmart,it is an android based application helps you find a place to park using live information from parking spaces around the city. ParkSmart app reads your smartphone GPS based location then directs you to your chosen parking space location. Using the slot allocation method we can book our own parking slot at any particular area/location. It is an efficient one for solving parking problems, which overcomes the traffic congestion also provides automated billing process.
This solution alone initiates a chain-reaction of benefits, from lesser traffic congestion to reduced fuel efficiency, in urban areas where parking is often painstaking.
